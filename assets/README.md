# Portfólio Profissional | Karina Lyra 🚀

Este repositório contém o código-fonte do portfólio profissional de **Karina Lyra**, desenvolvido como parte da trajetória de transição para a área de tecnologia (Projeto Versão 2.0). O projeto une uma sólida base jurídica e administrativa a novas competências em **Análise e Desenvolvimento de Sistemas (ADS)**.

## 🎯 Objetivo
O projeto visa apresentar de forma elegante e funcional a trajetória profissional da autora, destacando projetos práticos em desenvolvimento de software e design de interface (UI/UX), com foco em acessibilidade e performance.

## 🛠️ Tecnologias Utilizadas
O desenvolvimento foi realizado utilizando as melhores práticas de desenvolvimento web front-end:
- **HTML5:** Estruturação semântica para melhor SEO e acessibilidade.
- **CSS3:** Uso de Variáveis (Design System), Flexbox e CSS Grid para layout responsivo.
- **JavaScript:** Manipulação de DOM para interatividade (Carrossel e Validação de Formulário).
- **Font Awesome:** Integração de ícones vetoriais.

## 📂 Projetos em Destaque
1. **Sub Attack:** Jogo de ação desenvolvido em **Python/Pygame**, focado em lógica de colisões e gerenciamento de eventos.
2. **Game School:** Aplicação educativa gamificada, também desenvolvida em **Python/Pygame**, voltada para o aprendizado dinâmico.
3. **Vida Plus Saúde:** Protótipo de alta fidelidade desenvolvido no **Figma**, focado na experiência do usuário (UX) em sistemas hospitalares.

## ♿ Acessibilidade e UI/UX
O site foi projetado seguindo princípios de:
- **Navegação Semântica:** Uso de marcos (main, nav, section, article).
- **Acessibilidade:** Implementação de atributos ARIA e contraste adequado para leitura.
- **Design Responsivo:** Adaptabilidade para dispositivos móveis e desktops.
- **Interatividade:** Efeitos de hover com feedback visual (Neon Glow) e transições suaves.
